## 安装步骤

```
git clone git@github.com:openhnteam/analytics-admin-front.git      // 把模板下载到本地
yarn install         // 安装项目依赖，等待安装完成之后，安装失败可用 cnpm 或 yarn

// 运行
yarn start

// 执行构建命令，生成的dist文件夹放在服务器下即可访问
yarn build
```
